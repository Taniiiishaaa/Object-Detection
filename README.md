# 🎯 Object Detection and Tracking System  
### YOLOv8 + Deep SORT + Streamlit

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![OpenCV](https://img.shields.io/badge/OpenCV-Computer%20Vision-green)
![YOLOv8](https://img.shields.io/badge/YOLOv8-Ultralytics-orange)
![Streamlit](https://img.shields.io/badge/Streamlit-Interactive%20UI-red)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

A **real-time object detection and multi-object tracking system** built using **Ultralytics YOLOv8** and **Deep SORT tracking**.  
The project includes a **modern Streamlit dashboard** supporting **webcam, image, and video inputs** for interactive detection and tracking.

This project demonstrates practical **computer vision, deep learning, and real-time processing** skills.

# 🚀 Features

✔ **YOLOv8 Model Selection** (`n`, `s`, `m`, `l`, `x`)  
✔ **Real-time object detection**  
✔ **Deep SORT multi-object tracking**  
✔ **Webcam, image, and video input support**  
✔ **Interactive Streamlit GUI**  
✔ **Real-time FPS monitoring**  
✔ **Class-wise object count summary**  
✔ **Detection filtering by object classes**  
✔ **Confidence threshold adjustment**  
✔ **Detection log console**  
✔ **Download processed video output**  
✔ **Export detection summary as CSV**

---

# 🧠 Supported Object Classes

Based on the **COCO dataset**.

Examples include:

- `person`
- `car`
- `dog`
- `cat`
- `truck`
- `bicycle`
- `motorcycle`
- `bus`
- `bottle`
